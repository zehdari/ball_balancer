# ball_balancer

A 3-servo robot that balances a ping pong ball on a tiltable plate.

Using IsaacLab for training.

## Structure

**`Isaac/`** - Reinforcement learning training and playback in Isaac Lab. See [`Isaac/ball_balance_lab/README.md`](Isaac/ball_balance_lab/README.md) for setup and usage.

**`ros/`** - ROS2 workspace.
